# 📧 Email Tone Rewriter

Rewrite emails instantly with AI and transform them into professional, formal, friendly, or executive-level communication.

## 🚀 Overview

Email Tone Rewriter helps students, professionals, job seekers, and business users improve their emails with a single click.

Simply paste your email, select a tone, and let AI rewrite it into a polished and effective version while improving grammar, clarity, and professionalism.

---

## ✨ Features

* AI-powered email rewriting
* Multiple tone options
* Grammar correction
* Professional formatting
* Subject line suggestions

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/VishnuA-ai/Email-Tone-Rewriter.git
cd Email-Tone-Rewriter
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Create a `.env` file:

```env
GROQ_API_KEY=your_groq_api_key
```

---

## ▶️ Run Locally

```bash
streamlit run app.py
```

---

## 📖 How It Works

1. Paste your email.
2. Select your preferred tone.
3. Click **Rewrite Email**.
4. Receive:

   * Suggested Subject
   * Improved Email
   * Grammar Enhancements
5. Download the final version.
